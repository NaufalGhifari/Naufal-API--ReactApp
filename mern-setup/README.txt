## README.txt ##

- Client: Front-end, contains the react app (API call is made from here)
- Server: Back-end, contains the connection to a mongoDB Atlas (not used)

To setup: go to "client" folder and run "npm install node"
To start: go to client folder, run "npm start"

The main file is "client/src/App.jsx"

This app does NOT use: axios and bootstrap
